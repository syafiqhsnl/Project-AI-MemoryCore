## Meaningful Insights
- **The Empty Zip Mystery**: Discovered that n8n was producing empty GTFS zip files (0-6KB) during extraction. This systemic failure blocked the OTP build and highlighted the need for "Zero-Byte" validation in our sync workflows.

## Decisions Made
- **Reduced Heap**: Lowered GitHub Action Java heap from `6G` to `5G` to provide a safety buffer for the runner's operating system.
- **Split-Architecture**: Decided to keep the database on the old server to simplify migration and reduce initial load on the new VPS.
